# Galactic XCM config

[![npm version](https://img.shields.io/npm/v/@galacticcouncil/xcm-cfg.svg)](https://www.npmjs.com/package/@galacticcouncil/xcm-cfg)

Pre-configured routes for Hydration and Basilisk dApps. Maps assets and chains to transfer strategies.

## Installation

Install with [npm](https://www.npmjs.com/):

`npm install @galacticcouncil/xcm-cfg`

## Supported protocols

- **XCM**
- **Wormhole** (via MRL)
- **Snowbridge**
